#  ICO Smart Contract

A KYC-enabled ICO smart contract built using Solidity for secure and scalable token sales with multi-round pricing, referral rewards, oracle integration, and signature-based verification.

## Overview

ICO-V2 is a decentralized token sale platform designed to support secure and flexible fundraising on EVM-compatible chains. The contract includes multiple sale rounds, dynamic pricing, referral systems, KYC verification, and support for both native coin and token payments.

The project focuses on security, scalability, and smooth participation flow for users.

---

## Features

* Multi-round ICO system with dynamic token pricing
* KYC-based participation using signature verification
* Buy tokens using native coin or supported ERC20 tokens
* Referral reward mechanism
* Chainlink oracle integration for live price conversion
* AccessControl-based admin management
* Emergency admin controls
* Secure nonce protection to prevent replay attacks
* Event emission for tracking purchases and claims
* Gas-optimized Solidity implementation

## Smart Contract Modules

### ICO Sale Logic

Handles:

* Token sale rounds
* Token allocation
* Price updates
* Purchase validation
* User contributions

### KYC Verification

Users participate through signed messages generated by backend verification.

Includes:

* Signature validation
* Nonce protection
* Expiry checks

### Referral System

Supports:

* Referral tracking
* Reward distribution
* Referral statistics

### Oracle Integration

Uses Chainlink price feeds for:

* Native coin to USD conversion
* Dynamic token purchase calculation

### Admin Controls

Built using OpenZeppelin AccessControl.

Admin functionalities include:

* Start/stop ICO
* Update round pricing
* Withdraw funds
* Manage accepted tokens
* Update referral percentages
